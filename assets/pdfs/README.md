# PDF Study Guides

This folder contains downloadable PDF study guides and educational materials.

## Generated PDFs

The following PDF files are available for download:

1. **Algebra_Study_Guide.pdf** - Formulas, worked examples, and practice problems
2. **Math_Practice_Problems.pdf** - Algebra and geometry practice problems with answer key
3. **Chemistry_Formula_Sheet.pdf** - Essential chemistry formulas and common compounds
4. **Science_Experiment_Guide.pdf** - Step-by-step experiment instructions
5. **History_Timeline_Chart.pdf** - Major historical events and key concepts
6. **Vocabulary_Flashcard_Set.pdf** - Important vocabulary words and definitions
7. **Business_Fundamentals.pdf** - Key business terms and real-world examples
8. **Practice_Quiz.pdf** - Multiple-choice questions with answer key

## Generating PDFs

To generate all PDF files:

1. Open `generate-pdfs.html` in your web browser
2. Click the "Generate All PDFs" button
3. All PDF files will be automatically downloaded to this folder
4. The PDFs contain real, readable educational content

## PDF Content

Each PDF includes:
- Clear titles and headings
- Educational content (formulas, examples, problems)
- Bullet points and organized sections
- Answer keys where applicable
- Professional formatting

All PDFs are ready to use and contain actual educational content, not placeholders.
