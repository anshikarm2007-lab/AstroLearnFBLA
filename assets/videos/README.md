# Educational Videos

This folder contains educational video files in MP4 format.

## Video Files

The following video files should be placed in this directory:

1. `algebra-intro.mp4` - Introduction to Algebra (15:30)
2. `cell-biology.mp4` - Cell Biology Explained (22:45)
3. `ww2-overview.mp4` - World War II Overview (28:20)
4. `periodic-table.mp4` - Periodic Table Mastery (18:15)
5. `shakespeare-sonnets.mp4` - Shakespeare's Sonnets (25:10)
6. `calculus-fundamentals.mp4` - Calculus Fundamentals (35:00)

## Adding Videos

To add videos to the website:

1. Place MP4 video files in this `assets/videos/` folder
2. Ensure filenames match the names listed above
3. Videos should be in MP4 format for maximum browser compatibility
4. Recommended: Keep video file sizes reasonable for faster loading

## For Demo/Prototype

If you don't have actual video files, you can:
- Use short placeholder videos (even 5-10 second clips work for demo)
- Create simple educational content using screen recording
- Use free educational video resources (with proper attribution)

The video player will work with any valid MP4 file placed in this directory.
